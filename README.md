# ProjectRTC

## Install

It requires [node.js](http://nodejs.org/download/)

* git clone https://github.com/chiraggoti2016/webrtc.git
* cd webrtc/
* npm install
* npm start

The server will run on port 3000.
You can test it in the (Chrome or Firefox) browser at localhost:3000.
